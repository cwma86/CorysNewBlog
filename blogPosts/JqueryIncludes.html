Using Jquery to reduce duplication

This post will discuss how to use Jquery to reduce the amount of duplicate HTML throughout your website. This is done using the Jquery .load function to replace common blocks of HTML throughout your website

For example, I did this with the navigation pane, side bar, and footer of my website The navigation pane as shown here.

While this pane created with only about 25 lines of code (see nav.html below), the template I used as the basis for this site had those lines repeated across every page. As I am sure you can imagine, this quickly becomes a maintainability nightmare, where adding/editing something in the navigation pane requires identical edits on everyone of you websites pages.
